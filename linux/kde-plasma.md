# KDE Plasma 

## Dropshadow on snapped windows

In KDE When snapping windows - the window maintains a dropshadow. This causes the shadow to bleed to other displays. For example. having a 2monitors top/bottom. snapping a window on the top monitor will cause the shadow to bleed onto the bottom monitor. 

Disable this in the Window Decorations settings. 

Colours & Themes > Window Decorations > Edit Current theme

Check: "Draw border on maximised and tiled windows"

That will remove the drop shadow.
